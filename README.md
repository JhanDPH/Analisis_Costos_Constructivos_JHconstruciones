# 📊 Análisis de Costos Constructivos - JH Construcciones

## 🎯 Objetivo
Analizar costos, desviaciones y tiempos en proyectos de construcción.

## 📂 Dataset
Datos de proyectos con costos estimados vs reales.

## 📊 Análisis realizados
- Costos por fase
- Desviaciones
- Retrasos
- Impacto del clima

## 🧠 Insights
- Alto sobrecosto en fases iniciales
- Impacto del clima en retrasos

## 🛠️ Tecnologías
- Python
- Pandas
- Matplotlib
- Streamlit

## 🚀 Cómo ejecutar
```bash
streamlit run dashboard/app.py
